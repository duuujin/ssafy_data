# 📚 Pandas & 기초 통계 핵심 요약본

## 1. Pandas & NumPy 핵심 (데이터 핸들링)

### 🔹 데이터 구조
* **Series**: 1차원 배열. (Index + Value)
* **DataFrame**: 2차원 표 형식. (Row + Column)
* **ndarray**: NumPy의 다차원 배열 객체. Pandas 연산의 기초가 됨.

[Image of the structure of a Pandas DataFrame and Series]

### 🔹 데이터 로드 및 확인
* `pd.read_csv('file.csv')`: CSV 파일 불러오기.
* `df.head(n)` / `df.tail(n)`: 상위/하위 데이터 확인.
* `df.shape`: (행, 열) 크기 확인.
* `df.info()`: 컬럼명, 데이터 타입, 결측치(Null) 여부 확인.
* `df.describe()`: 수치형 데이터의 기초 통계량(평균, 표준편차, 사분위수 등) 요약.
* `df.value_counts()`: 행을 기준으로 중복된 행이 얼마나 있는지 확인. Null값 무시하고 결과값 리턴(기본 인자값 : dropna = Ture) -> NaN포함은 dropna = False
* `df.sort_values()`: 특정 컬럼 기준으로 정렬
* `df.sample(5)`: DataFrame의 랜덤하게 일부 데이터 추출.
* `df.drop('col', axis=1)` : 특정 열(Column) 삭제.
    - axis = 0 로우(행), axis = 1 컬럼(열)
    - 로우 삭제 시 axis 생략 가능



### 🔹 데이터 선택 및 필터링
| 방식 | 기준 | 특징 |
| :--- | :--- | :--- |
| **`loc[]`** | 명칭 (Label) | 인덱스 이름, 컬럼명으로 접근 (`df.loc[0, 'Name']`) |
| **`iloc[]`** | 위치 (Integer) | 정수 위치 인덱스로 접근 (`df.iloc[0, 1]`) |

* **Boolean Indexing**: 조건에 맞는 행 추출. (예: `df[df['age'] >= 20]`)
* **다중 조건**: `&` (AND), `|` (OR) 기호 사용.

### 🔹 데이터 전처리 및 가공
* **결측치 처리**: `isna()` (확인), `dropna()` (제거), `fillna()` (채우기).
* **정렬**: `sort_values(by='컬럼', ascending=True/False)`.
* **그룹화**: `groupby('기준컬럼').집계함수()` 또는 `.agg(['mean', 'sum'])`.
* **병합**: `merge()` (공통 키 기준 Join), `concat()` (단순 연결).
* **수정**: `apply(lambda x: ...)` (일괄 적용), `replace()` (값 교체).

## 2. 기초 통계 핵심 (개념 및 가설 검정)

### 🔹 표본 추출과 분포
* **모집단(Population) vs 표본(Sample)**: 전체 집단에서 조사 대상을 추출.
* **표본 편향(Sampling Bias)**: 특정 집단만 추출되어 결과가 왜곡되는 것.
* **중심극한정리(CLT)**: 표본 크기가 충분히 크면 표본 평균의 분포는 정규분포를 따름.
* **정규분포**: 평균($\mu$)을 중심으로 대칭인 종 모양 분포.
    * ±1 표준편차: 약 68% 데이터 존재.
    * ±2 표준편차: 약 95% 데이터 존재.



### 🔹 가설 검정 (Hypothesis Testing)
1. **귀무가설 ($H_0$)**: "차이가 없다", "효과가 없다" (지키려는 가설).
2. **대립가설 ($H_1$)**: "차이가 있다", "효과가 있다" (증명하려는 가설).
3. **p-value (유의확률)**: 귀무가설이 참일 때, 현재 결과가 나올 확률.
    * $p < 0.05$: 귀무가설 기각 $\rightarrow$ **통계적으로 유의미함.**
    * $p \ge 0.05$: 귀무가설 채택 $\rightarrow$ **우연일 가능성이 높음.**
4. **t-test**: 두 집단의 평균 차이가 유의미한지 확인하는 검정 도구.
