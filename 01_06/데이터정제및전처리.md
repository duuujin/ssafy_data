# 데이터의 처리
- 데이터 수집이 완료되면 데이터를 모델에 넣기전에 알맞게 가공하는 과정
- 분석 결과의 질을 높이기 위해 데이터를 변형하는 과정
- 어떤 전처리를 적용하느냐에 따라 원본 데이터가 다르게 변형되므로, 결과도 달라질 수 있음

# 데이터의 처리 방법
- 데이터의 형식 맞추기
- 빈 칸 채우기
- 데이터 열 추가(연관 데이터 추가)
- 데이터 열 추가(이상치 제거)

# missing 데이터 처리
- isna() : 데이터프레임 또는 배열에서 결측값(NaN)이 있는지 여부를 True/False 반환
- fillna() :  결측값(NaN)을 지정한 값으로 대체(채움)
- dropna() : 결측값(NaN)이 포함된 행 또는 열을 제거
