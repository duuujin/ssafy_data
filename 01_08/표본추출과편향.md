# 모집단과 표본의 개념
- 모집단 : 연구 대상이 되는 전체 집단
- 표본 : 모집단에서 일부를 선택한 것

# 표본 조사의 필요성
- 인구 조사 vs 표본 조사
    - 인구조사(Census) -> 모집단 조사
    - 전체 국민을 조사하는 방법
    - 가장 정확하지만, 시간과 비용이 많이 소모
    - 예 : 대한민국의 인구 총 조사(모든 국민 조사)

# 표본 추출
- 모집단에서 일부를 선택(추출)하는 과정

## 표본 추출의 방법
- 단순 무작위 추출
    - 모든 사람이 동일한 확률로 뽑힘
    - 예 : 모자 속에서 랜덤하게 종이 한 장 뽑기
- 층화추출
    - 모집단을 그룹(층)으로 나눈 후, 각 층에서 표본 추출
    - 예 : 학년별 반 대표 한 명씩 뽑기
- 계통추출
    - 모집단을 일정 간격으로 나누어, 표본을 추출
    - 예 : 첫 번째 학생을 랜덤으로 선택한 후, 5명마다 한 명씩 조사
- 군집추출
    - 모집단을 여러 그룹(군)으로 나눈 후, 몇 개의 군을 랜덤 선택
    - 예 : 한 반을 통째로 뽑아 조사
- 편의추출
    - 쉽게 접근 할 수 있는 표본을 선택(비추천)
    - 예 : 길거리에서 가까운 사람들만 설문조사
